Base URL

/jobs
Endpoints
1. List Jobs Records
Method: GET
Endpoint: /jobs
Description: Retrieves a list of job records, 

2. View Job Record
Method: GET
Endpoint: /jobs/view/{recid}
Path Parameters:
recid: The unique identifier of the job record to view.
Description: Retrieves a specific job record by its ID.

3. Add Job Record
Method: POST
Endpoint: /jobs/add
Request Body:
title: The title of the job (required).
description: A description of the job (required).
skills: A comma-separated list of skills required for the job (required).
Description: Adds a new job record to the database.

4. Get Job Record for Edit
Method: GET
Endpoint: /jobs/edit/{recid}
Path Parameters:
recid: The unique identifier of the job record to edit.
Description: Retrieves a specific job record for editing.

5. Update Job Record
Method: POST
Endpoint: /jobs/edit/{recid}
Path Parameters:
recid: The unique identifier of the job record to update.
Request Body:
title: The title of the job (optional).
description: A description of the job (optional).
skills: A comma-separated list of skills required for the job (optional).
Description: Updates an existing job record in the database.

6. Delete Job Record
Method: GET
Endpoint: /jobs/delete/{recid}
Path Parameters:
recid: The unique identifier(s) of the job record(s) to delete, separated by commas for multiple records.
Description: Deletes one or more job records from the database.

7. Get Recommended Candidates for a Job
Method: GET
Endpoint: /jobs/{job_id}/candidates
Path Parameters:
job_id: The unique identifier of the job for which to recommend candidates.
Description: Retrieves a list of recommended candidates based on the skills required for the specified job. The top 3 candidates are returned along with a generated professional summary for each.


Error Handling
The API returns appropriate HTTP status codes for different scenarios:

200 OK: Successful request.
400 Bad Request: Validation errors or malformed requests.
404 Not Found: Resource not found (e.g., job record does not exist).
500 Internal Server Error: Server-side errors.


Example Requests
List Jobs
GET /jobs/

View Job
GET /jobs/view/1

Add Job
POST /jobs/add
Content-Type: application/json

{
  "title": "Software Engineer",
  "description": "Develop and maintain software applications.",
  "skills": "Java, JavaScript, SQL"
}

Update Job
POST /jobs/edit/1
Content-Type: application/json

{
  "description": "Develop and maintain software applications with a focus on performance."
}

Delete Job
GET /jobs/delete/1

Get Recommended Candidates
GET /jobs/1/candidates
